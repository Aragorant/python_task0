# python_task2
Програма підтримує основні (+, -, /, *) та додаткові (cos, sin, tan, ctg, log, ln, %) функції калькулятора.
Також у програму можна водити дробові числа. Кнопка 'c' видалить всі символи, а кнопка 'del' видаляє по одному символу.
